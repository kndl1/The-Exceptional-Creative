How To Use It

The toolkit was created for designers who own Adobe® Creative Suite®, particularly InDesign®. These files are prepared for CS5 (INDD) and earlier versions (IDML).  The Docket spreadsheet has been saved for Excel 2008 for Mac and Excel 2007 for Windows (.xlsx) as well as Excel 98 through Excel 2004 for Mac and Excel 97 through Excel 2003 for Windows (.xls).
